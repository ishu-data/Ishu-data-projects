# 📚 Bookstore Management System (PostgreSQL)
## 📌 Overview

- This project demonstrates the design and implementation of a relational database for a bookstore using PostgreSQL.

- It includes table creation scripts and SQL queries to solve real-world business problems such as customer analysis, order tracking, inventory management, and sales reporting.


## 🛠 Technologies Used

- PostgreSQL
- SQL


## 🗄 Database Tables

- Books
- Customers
- Orders


## 📊 SQL Concepts Covered

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions
- INNER JOIN
- LEFT JOIN
- Subqueries
- CASE Statements
- Common Table Expressions (CTEs)
- Window Functions

## 📈 Business Problems Solved

- Retrieve customer information
- Analyze book sales
- Find top-selling books
- Calculate total revenue
- Track customer orders
- Generate inventory insights

## 🎯 Skills Demonstrated

- Relational Database Design
- Query Optimization
- Data Analysis using SQL
- Business Intelligence Queries
- PostgreSQL Database Management
